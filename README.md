智能停车场管理系统
实现功能：模拟停车位，利用超声波测距装置判断车位是否为空，系统显示屏空车位信息更新（停车场总车位与剩余车位个数信息），空车位上方LED亮起提示司机此处可以停车，当用户停车入位时开始系统开始计时，离开时系统自动计算停车费用，用户可以通过手机登录停车系统查询是否有空车位并查询停车时间，费用等信息。
功能模块划分：超声模块，LCD显示屏，单片机核心控制，数据网络互传模块，手机APP模块等。
建议团队人数：6-8人
课程进度安排：需求分析+方案的论证（硬件芯片或型号的选型）+团队成员的分工+模块间软硬件接口的定义+流程图的绘制+代码的书写和调试+功能的完善改进+报告的书写。
完成指标要求：
（1）超声测距模块，单片机核心控制模块；
（2）显示屏显示车位信息；
（3）能够远程实时查看停车场车位情况以及停车费用等信息查询。
最终需要提交的材料：报告一份（项目的综述，软硬件的介绍说明，各自的分工说明，遇到的问题及解决的办法，各自对本课程的体会感悟，对课程的建议），实现的代码，相应的原理图和流程图。
